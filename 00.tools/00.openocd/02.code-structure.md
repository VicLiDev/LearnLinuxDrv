## 2. 代码结构总览

### 2.1 顶层目录

```
openocd/
├── src/           # 核心源码
├── jimtcl/        # 内嵌的 Jim Tcl 解释器
├── tcl/           # TCL 配置脚本（板级配置、目标配置）
├── contrib/       # 贡献代码和示例
├── testing/       # 测试套件
├── doc/           # 文档
├── tools/         # 工具脚本
├── build-aux/     # 构建辅助文件
└── LICENSES/      # 许可证文件
```

### 2.2 src/ 目录结构

```
src/
├── main.c                    # 程序入口 (调用 openocd_main)
├── openocd.c                 # 核心启动逻辑、init 命令处理
│
├── helper/                   # 基础工具库
│   ├── command.c/h           # 命令注册与调度系统
│   ├── configuration.c       # 配置文件处理
│   ├── options.c             # 命令行参数解析
│   ├── log.c                 # 日志系统
│   ├── binarybuffer.c        # 二进制缓冲区操作
│   ├── replacements.c        # 平台兼容层
│   └── util.c                # 通用工具函数
│
├── jtag/                     # JTAG/SWD 调试协议层
│   ├── core.c/h              # JTAG 核心 (TAP 管理、扫描链)
│   ├── interface.c/h         # JTAG 接口抽象
│   ├── adapter.c/h           # 适配器管理 (时钟、复位)
│   ├── swd.c/h               # SWD 协议实现
│   ├── tcl.c                 # JTAG TCL 命令
│   ├── commands.c            # JTAG C 命令
│   ├── hla/                  # 高层访问接口 (ST-Link 等)
│   └── drivers/              # 硬件驱动 (30+ 种)
│       ├── ftdi.c            #   FTDI 芯片驱动
│       ├── jlink.c           #   J-Link 驱动
│       ├── stlink_usb.c      #   ST-Link USB 驱动
│       ├── cmsis_dap_usb.c   #   CMSIS-DAP 驱动
│       ├── openjtag.c        #   OpenJTAG 驱动
│       └── ...
│
├── transport/                # 传输协议抽象层
│   └── transport.c/h         # JTAG/SWD 传输抽象
│
├── target/                   # 目标处理器架构层
│   ├── target.c/h            # 目标管理核心
│   ├── target_type.h         # target_type 接口定义
│   ├── register.c/h          # 寄存器系统
│   ├── breakpoints.c/h       # 断点管理
│   ├── watchpoints.c/h       # 观察点管理
│   ├── image.c               # 二进制镜像加载
│   ├── algorithm.c           # 运行算法 (Flash 烧写等)
│   ├── armv7m.c              # ARMv7-M 公共层
│   ├── cortex_m.c            # Cortex-M 实现
│   ├── cortex_a.c            # Cortex-A 实现
│   ├── arm7tdmi.c            # ARM7TDMI 实现
│   ├── arm9tdmi.c            # ARM9TDMI 实现
│   ├── riscv/                # RISC-V 实现
│   ├── mips32.c              # MIPS32 实现
│   └── ...
│
├── flash/                    # Flash 编程系统
│   ├── nor/                  # NOR Flash 驱动
│   │   ├── core.c/h          #   NOR Flash 框架
│   │   ├── driver.h          #   flash_driver 接口
│   │   ├── stm32f1x.c        #   STM32F1 Flash 驱动
│   │   ├── stm32h7x.c        #   STM32H7 Flash 驱动
│   │   ├── stm32l4x.c        #   STM32L4 Flash 驱动
│   │   ├── at91samd.c        #   Atmel SAMD Flash 驱动
│   │   ├── efm32.c           #   EFM32 Flash 驱动
│   │   └── ...
│   └── nand/                 # NAND Flash 驱动
│       ├── core.c/h          #   NAND Flash 框架
│       └── ...
│
├── server/                   # 服务端系统
│   ├── server.c/h            # 服务器框架
│   ├── gdb_server.c          # GDB 远程协议服务器
│   ├── telnet_server.c       # Telnet 交互服务器
│   ├── tcl_server.c          # TCL 服务器
│   └── rtt_server.c          # RTT (Real-Time Transfer) 服务器
│
├── rtos/                     # RTOS 感知支持
│   ├── rtos.c                # RTOS 框架
│   ├── freertos.c            #   FreeRTOS 支持
│   ├── threadx.c             #   ThreadX 支持
│   ├── zephyr.c              #   Zephyr 支持
│   └── ...
│
├── rtt/                      # SEGGER RTT 支持
├── svf/                      # SVF 文件解析 (边界扫描)
├── xsvf/                     # XSVF 文件解析
└── pld/                      # 可编程逻辑器件支持
```

### 2.3 整体架构分层

```
┌───────────────────────────────────────────────┐
│              用户接口层                       │
│   GDB Server  │  Telnet Server  │  TCL Server │
├───────────────────────────────────────────────┤
│              命令调度层                       │
│        command.c (TCL 解释器集成)             │
├──────────┬──────────┬──────────┬──────────────┤
│ Target层 │ Flash层  │ RTOS层   │  其他功能层  │
├──────────┴──────────┴──────────┴──────────────┤
│           传输协议抽象层                      │
│         transport.c (JTAG/SWD)                │
├───────────────────────────────────────────────┤
│           JTAG/SWD 核心协议层                 │
│     core.c │ swd.c │ adapter.c │ interface.c  │
├───────────────────────────────────────────────┤
│           硬件驱动层                          │
│    ftdi │ jlink │ stlink │ cmsis_dap │ ...    │
└───────────────────────────────────────────────┘
```

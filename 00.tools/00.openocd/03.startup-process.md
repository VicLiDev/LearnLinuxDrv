```c
COMMAND_HANDLER(handle_init_command)
```

### 3.4 init 阶段的模式切换

OpenOCD 有两种命令执行模式（`src/helper/command.h:39`）：

```c
enum command_mode {
    COMMAND_EXEC,    // 执行模式：init 之后的阶段
    COMMAND_CONFIG,  // 配置模式：init 之前的阶段
    COMMAND_ANY,     // 任意模式均可执行
};
```

- 初始为 `COMMAND_CONFIG`，可以解析配置文件
- `init` 过程中切换为 `COMMAND_EXEC`，启用完整调试功能
- 某些命令只能在特定模式下执行

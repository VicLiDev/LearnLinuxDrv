## 7. Flash 编程系统

### 7.1 架构概述

```
用户命令 (flash write_image / flash erase_sector)
    ↓
Flash 核心框架 (src/flash/nor/core.c)
    ├── 地址映射到扇区
    ├── 扇区对齐处理
    ├── 镜像格式解析 (ELF/IHEX/BIN)
    └── 调用驱动实现
         ↓
具体 Flash 驱动 (stm32f1x.c / stm32h7x.c / ...)
    ├── 解锁 Flash
    ├── 擦除扇区
    ├── 编程写入
    ├── 校验
    └── 加锁/保护
```

### 7.2 核心数据结构

**Flash Bank** (`src/flash/nor/core.h:75`)：
```c
struct flash_bank {
    char *name;               // Flash 名称
    struct target *target;    // 所属目标
    const struct flash_driver *driver;  // Flash 驱动
    void *driver_priv;        // 驱动私有数据
    target_addr_t base;       // 基地址
    uint32_t size;            // 总大小
    int chip_width;           // 芯片宽度
    int bus_width;            // 总线宽度
    int num_sectors;          // 扇区数量
    struct flash_sector *sectors;  // 扇区描述数组
    bool probed;              // 是否已探测
    int default_padded_value; // 填充值
};
```

**Flash 扇区** (`src/flash/nor/core.h:28`)：
```c
struct flash_sector {
    target_addr_t offset;     // 扇区内偏移
    uint32_t size;            // 扇区大小
    bool is_erased;           // 是否已擦除
    bool is_protected;        // 是否受保护
};
```

**Flash 驱动接口** (`src/flash/nor/driver.h:39`)：
```c
struct flash_driver {
    const char *name;

    // 探测和配置
    int (*flash_bank_command)(struct command_context *cmd_ctx);
    int (*probe)(struct flash_bank *bank);

    // 基本操作
    int (*protect_check)(struct flash_bank *bank);
    int (*protect)(struct flash_bank *bank, int set, int first, int last);
    int (*erase)(struct flash_bank *bank, int first, int last);

    // 高级操作
    int (*write)(struct flash_bank *bank, const uint8_t *buffer,
                uint32_t offset, uint32_t count);
    int (*read)(struct flash_bank *bank, uint8_t *buffer,
               uint32_t offset, uint32_t count);
    int (*verify)(struct flash_bank *bank, const uint8_t *buffer,
                 uint32_t offset, uint32_t count);

    // 信息查询
    int (*info)(struct flash_bank *bank, char *buf, int buf_size);
    int (*erase_check)(struct flash_bank *bank);
    int (*protect_check)(struct flash_bank *bank);

    const char *usage;
};
```

### 7.4 镜像格式支持

`src/target/image.c` 支持多种镜像格式：

| 格式      | 说明                | 扩展名     |
|-----------|---------------------|------------|
| ELF       | 可执行链接格式      | .elf, .axf |
| Intel HEX | Intel 十六进制格式  | .hex       |
| SRecord   | Motorola S 记录格式 | .srec      |
| Binary    | 原始二进制          | .bin       |

### 7.5 算法执行

Flash 编程通常使用 **在目标芯片上运行的算法**：

```
1. 将 Flash 编程算法代码下载到目标 RAM (working_area)
2. 设置算法参数 (源地址、目标地址、大小)
3. 设置入口/退出断点
4. 让目标执行算法
5. 等待算法完成 (到达退出断点)
6. 读取结果
```
